# goapk
将go代码打包为android的apk包

# 打包apk的环境:
 * [ubuntu 20.04](https://ubuntu.com/)
 * [android ndk r23b](https://developer.android.com/ndk/downloads)
 * [go 1.8](https://golang.google.cn/)
 * [fyne](https://fyne.io/)

# 输出apk文件的大小:
  * 单个平台为 16MB
  * 全平台 arm、arm64、x86、x86_64 合集为 62MB
