# goapk
将go代码打包为android的apk包

# 打包apk的环境:
 * [archlinux](https://archlinux.org/)
 * [android sdk 29, ndk](https://developer.android.com/studio)
 * [go 1.8](https://golang.google.cn/)
 * [fyne](https://fyne.io/)

# 输出apk文件的大小:
  * 单个平台为 16MB
  * 全平台 arm、arm64、x86、x86_64 为 62MB
